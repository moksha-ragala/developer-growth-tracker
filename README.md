# Developer Growth Tracker

A full-stack platform that helps developers track technical growth through GitHub analytics, coding consistency, skill gap analysis, and personalized learning recommendations.

## Tech Stack

- Java
- Spring Boot
- Spring Security
- MySQL
- React
- Azure

## Planned Features

- User Authentication
- GitHub Analytics
- Developer Score
- Skill Gap Analysis
- Resume Analyzer
- Learning Roadmap

## Architecture

Frontend (React)
       ↓
REST APIs
       ↓
Spring Boot Backend
       ↓
MySQL Database
